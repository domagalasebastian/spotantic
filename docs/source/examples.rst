Examples
========

The `examples/` directory in the repository contains small runnable
examples demonstrating common initialization and usage patterns.

Running an example
-------------------

From the repository root run Python on the example script. For example:

.. code-block:: bash

   python examples/create_client.py

Some examples read configuration from environment variables. Check the
top of each example script for instructions.

Refer to the `quickstart` page for a minimal programmatic example and
to the client and auth reference pages for full API details.
